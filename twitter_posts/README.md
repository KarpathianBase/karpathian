## Honest-scale caveat

These posts intentionally name the limits in 7/7: synthetic-data 20-step proxy_cpu_smoke, testnet not mainnet, deltas are small in absolute terms. The harness was built to prove the protocol fires distributed, not to set SOTA. Burying that would be cope.
